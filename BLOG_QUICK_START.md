# 🚀 Quick Start - Publishing Your First Blog Post

## ⚡ 5-Minute Guide

### 1. Login to Admin
Go to: `http://localhost:3000/en/admin/login`

### 2. Go to Blog Posts
Click **"Blog Posts"** in the sidebar

### 3. Create New Post
Click **"New Post"** button (top right)

### 4. Write Your Post

**English Tab:**
- ✍️ **Title**: "My First Blog Post" (dark text, easy to read!)
- ✍️ **Content**: Write your post using Markdown
- ✍️ **Excerpt**: "A brief summary of my first post"

**Post Settings:**
- 📁 **Category**: Choose one (Journey, Insights, etc.)
- 🏷️ **Tags**: Add tags separated by commas

### 5. Upload Images (Optional)

**Cover Image:**
- Click **"Upload Cover Image"**
- Select image (max 5MB)
- Wait for upload ✅

**Content Images:**
- Click **"Insert Image"** button above content
- Select image
- Markdown is auto-copied!
- Paste it in your content

### 6. Publish!

Look for the **Publication Status** box:

**Save as Draft (Orange Box):**
- Keep switch OFF
- Click "Create Post"
- Post is saved but not visible

**Publish Now (Green Box):**
- Toggle switch ON ✅
- Click "Create Post"
- Post goes live!

## 🔄 Publication Workflow

### Option 1: Draft → Publish
1. Write post
2. Save as **Draft** (switch OFF, orange box)
3. Review and edit
4. When ready: Toggle **Published** (switch ON, green box)
5. Save changes

### Option 2: Publish Immediately
1. Write post
2. Toggle **Published** (switch ON, green box)
3. Click "Create Post"
4. Post goes live!

### Option 3: Featured Post
1. Create/edit post
2. Toggle **Published** ON (green box)
3. Toggle **Featured** ON (⭐)
4. Post shows on homepage!

## 📸 Image Upload Workflow

### Cover Image
1. Scroll to **Post Settings**
2. Click **"Upload Cover Image"**
3. Choose image
4. See progress bar
5. Preview appears with ❌ remove button

### Content Images
1. Click **"Insert Image"** above content field
2. Choose image
3. Wait for upload
4. **Markdown auto-copied to clipboard!** 📋
5. Paste (Ctrl+V or Cmd+V) in your content
